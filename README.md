## 사용한 기술
+ PhotoShop
+ Illustrator
+ Figma
+ HTML5
+ CSS3
+ ECMAScript

## 구현 기능
+ 타이핑 되는 듯한 효과를 적용한 텍스트
+ 클릭 이벤트에 의한 화면 이동
+ 클릭 이벤트에 반응해 보여지거나 사라지는 배경
+ 화면 이동시 마치 전기 신호가 이동하는 듯 보이는 
+ Next/Prev 버튼과 pagenation이 연동되는 Slide
+ 곡선(path)을 따라 물체가 이동하는 애니메이션
+ 미니맵 형식의 네비게이션
+ 네비게이션 클릭시 지정된 경로에 따라 움직이는 화면
+ 반응형 적용
+ 아코디언 스타일 메뉴(반응형 페이지)
